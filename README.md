# trycmd-example

```console
$ trycmd-example --help
Simple program to greet a person

Usage: trycmd-example [OPTIONS] --name <NAME>

Options:
  -n, --name <NAME>    Name of the person to greet
  -c, --count <COUNT>  Number of times to greet [default: 1]
  -h, --help           Print help
  -V, --version        Print version

```
